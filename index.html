🔴 ROI Tracker
Connect your Google Sheet to get started
One-time setup — ~5 minutes
Your data will live in Google Sheets forever. Open this file from any browser and your data loads automatically.
1
Go to sheets.google.com → create a new blank spreadsheet → name it ROI Tracker
2
Click Extensions → Apps Script. Delete all existing code. Paste the entire Code.gs file contents. Press Ctrl+S to save.
3
Click Deploy → New deployment. Click the ⚙️ gear → select Web app. Set Execute as: Me and Who has access: Anyone. Click Deploy.
4
Google will ask to authorize — click Authorize access → choose your account → if you see "Google hasn't verified this app" click Advanced → Go to ROI Tracker (unsafe) → Allow. This is your own script — it's safe.
5
Copy the Web App URL (looks like https://script.google.com/macros/s/…/exec) and paste it below.
URL is saved in your browser. You won't need to enter it again.
